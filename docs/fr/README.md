# Documentation de morfAnalytics (français)

Moteur d'analyse de l'écosystème morfSystem : collecte incrémentale de
l'historique d'un équipement, analyses météorologiques et climatologiques,
page web. L'équipement reste la source de vérité ; ce service n'en lit qu'une
copie.
